# Gerador de Números Aleatórios

Este projeto é uma aplicação web funcional desenvolvida para realizar sorteios numéricos dentro de um intervalo customizável. O foco principal foi a aplicação de **lógica de programação** e **manipulação dinâmica do DOM**, demonstrando como conceitos matemáticos são traduzidos para o ecossistema JavaScript.

## 🚀 Funcionalidades

* **Sorteio Inclusivo:** Diferente de implementações básicas, este algoritmo garante que tanto o valor **mínimo** quanto o **máximo** definidos tenham a mesma probabilidade de serem sorteados.
* **Interface Modern Dark:** Design inspirado na paleta de cores do GitHub, proporcionando uma experiência visual limpa e profissional.
* **Design Responsivo:** Interface totalmente adaptável, garantindo usabilidade em dispositivos móveis, tablets e desktops.
* **Feedback de Interação (UX):** Uso de pseudo-classes CSS para efeitos de clique (`scale`) e estados de foco nos campos de entrada.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e acessibilidade.
* **CSS3:** Estilização avançada com **Flexbox**, variáveis de cores e transições suaves.
* **JavaScript (ES6+):** Lógica algorítmica e manipulação de eventos em tempo real.
